#### 笔记内容

- 计算机基础知识笔记
- 软件开发笔记
- 一些有用的工具笔记
- 大学部分课程笔记

#### 图片管理

- 采用了 PicGo 客户端和 Github 图床管理笔记中的图片
- 借用了 jsdelivr 的 CDN 加速

#### 笔记格式

采用了 Markdown 为主体文件格式，在其中使用 $\LaTeX{}$ 书写数学公式。

#### 笔记软件

本仓库笔记书写采用的编辑器为 Typora，使用 Typora 阅读体验可能会更好。
