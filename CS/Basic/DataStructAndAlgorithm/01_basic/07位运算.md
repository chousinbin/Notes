## 常用操作

### 1.求$x$进制的从右数的第$k$位数字 

```c++
x >> k-1 & 1;
```

#### 例:求十进制数18的二进制数的最后一个数是多少 

```c++
18 >> 0 & 1;
```

### 2.返回$x$的最后一位1所对应的值

```c++
int Lowbit(int x)
{
    return x & -x;
}
```

```c++
X & -X = X & (~X+1)
```

#### 例如：

下面的$1010$、$10$、$101000$、$1000$均为二进制数

```c++
lowbit(1010)=10;
lowbit(101000)=1000;
```

#### 过程：

```c++
       x = 1010...100...0
      ~x = 0101...011...1
    ~x+1 = 0101...100...0

x&(~x+1) = 0000...100...0
```



## 二进制中 1 的个数

### 暴力

```cpp
#include <bits/stdc++.h>
using namespace std;

const int N = 1e5 + 10;
int a[N];
 
int main()
{
	int n;
	cin >> n;
	for(int i = 0; i < n; i++) cin >> a[i];
	
	for(int i = 0; i < n; i++)
	{
		int res = 0;
		while(a[i])
		{
			res += a[i] & 1;
			a[i] >>= 1;
		}
		cout << res << ' ';
	}
	return 0;
} 
```
