 

## 编辑快捷键

| 功能                     | 快捷键         |
| ------------------------ | -------------- |
| 删除当前行               | ctrl + Y       |
| 复制当前行               | ctrl + alt + d |
| 补全代码                 | alt + /        |
| 添加注释/取消注释        | ctrl + /       |
| 导入该行所需要的类       | alt + enter    |
| 运行程序                 | alt + R        |
| 格式化代码               | ctrl + alt + L |
| 光标在方法名上定位到方法 | ctrl + B       |
| 自动生成构造方法         | alt + insert   |
| 自动分配变量名           | .var           |
