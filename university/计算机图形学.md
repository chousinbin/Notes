## survey

computer graphics: a computer is used to create a picture（图形）

#### 图形

- 本质：矢量
- 存储：程序
- 放大：不失真
- 显示：需要扫描转换

#### 图像

- 本质：标量
- 存储：二维矩阵
- 放大：失真
- 显示：直接显示

#### history

- 50s CRT显示器
- 60s Bezier曲线
- 70s 消隐算法、光照模型、颜色模型
- 90s mid
  - DOS -> Windows
  - Web / www / HTTP
  - Java
